# POSTA AUTO

Aplicación para organizar el uso de vehículos familiares.

## Objetivo
Permitir que familias administren uno o varios vehículos de forma simple, con:
- Reserva de vehículos
- Agenda compartida
- Vencimientos (seguro, VTV, licencia)
- Mantenimiento
- Historial de uso
- Notificaciones
- Estadísticas

Proyecto en desarrollo.
